# RoboFriends 機器人朋友們  

[Demo](https://boycany.github.io/robofriends/) 

## 作品說明

串接 JSONplaceholder 的 Free fake API 獲得使用者資料
> https://jsonplaceholder.typicode.com/

串接robohash.org的 API，將使用者資料轉換成機器人圖像
> https://robohash.org/

## 系統說明

框架：React <br>
樣式：Tachyons https://tachyons.io/

* React 版本 `v17.0.2`
* tachyons版本: `v4.12.0`
